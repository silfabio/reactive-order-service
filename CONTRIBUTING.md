# Contributing

We welcome contributions to the Reactive Order Service project!

## How to Contribute

1. **Fork the repository.**
2. **Create a new branch** for your feature or bug fix: `git checkout -b feature/your-feature-name` or `bugfix/your-bug-fix`.
3. **Make your changes.**
4. **Ensure all tests pass:** `./gradlew test`.
5. **Submit a pull request** with a clear description of your changes.

## Code Style

This project uses the standard Kotlin coding conventions. Please ensure your code is formatted correctly before submitting a pull request.

## Reporting Bugs

If you find a bug, please open an issue and provide as much detail as possible, including steps to reproduce it.
